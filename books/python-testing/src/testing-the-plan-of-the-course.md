# The plan of this course

1. Learn how to write unittests for simple functions.
1. Deal with special problems.
    * How to deal with randomness?
    * How to deal with time?
    * How to deal with reported bugs that we cannot fix now?
    * How to deal with setup and teardown of the environments?
1. Then get see what to do if we already have a working application that is hard to test.
